О приложении

VoxMind — это революционный подход к ведению дневника с использованием искусственного интеллекта

Что такое VoxMind?

VoxMind — это умный голосовой дневник, который использует искусственный интеллект для анализа ваших мыслей и эмоций. Просто говорите о своём дне естественно, а приложение автоматически преобразует речь в текст и проведет глубокий психологический анализ.

Ключевые преимущества:

  • Записывайте голосом — не нужно печатать
  • ИИ анализирует эмоции и настроение
  • Получайте персональные инсайты
  • Все данные хранятся локально на устройстве
  • Работает онлайн и офлайн

Технологии будущего

VoxMind использует передовые технологии искусственного интеллекта:

  • Распознавание речи на русском языке
  • Нейросетевой анализ эмоционального состояния
  • Определение 2+ эмоций и 2+ тем
  • Локальное машинное обучение

Основные функции

Все что нужно для ведения современного дневника

Голосовая запись

Записывайте свои мысли голосом в любое время. Автоматическое распознавание речи на русском языке.

Анализ эмоций

ИИ определяет ваши эмоции (радость, грусть, тревога, мотивация) и дает обратную связь.

Аналитика

Еженедельные отчеты о вашем эмоциональном состоянии, графики настроения и тренды.

Конфиденциальность

Все данные хранятся локально на устройстве. Ваши мысли остаются только с вами.

История записей

Удобный календарь для просмотра записей по дням и поиска по ключевым словам.

Инсайты

Персональные рекомендации и инсайты для самопознания и личностного роста.

Инструкция по установке

Для пользователей:
  1. Нажмите кнопку "Скачать VoxMind APK" выше
  2. Разрешите установку из неизвестных источников в настройках Android
  3. Откройте скачанный файл и нажмите "Установить"
  4. Запустите VoxMind и начните использовать!
Для разработчиков:
  1. Клонируйте репозиторий с GitHub
  2. Откройте проект в Android Studio
  3. Соберите APK через Build → Build Bundle(s) / APK(s)
  4. Установите на устройство и протестируйте
Важно: При первой установке Android может предупредить об установке из неизвестных источников. Это нормально, так как приложение пока не опубликовано в Google Play.
